# 品牌诊断案例

---

## 案例1：新锐咖啡品牌（产品力与定位错位）

### 诊断信息

- 客单价30元，日均客流50-80人
- 客户反馈：好评"环境不错"，中评"味道中规中矩"，差评"没有特色"
- 客群：25-35岁白领女性，路过消费
- 品牌诉求："第三空间"
- 老板判断：问题是"知名度不够"，想砸钱推广

### 诊断链条

**第一层：客单价与期待**
- 客单价30元 → 期待情绪价值 > 产品价值

**第二层：客户反馈解读**
- 好评"环境不错" → 环境有价值
- 中评"味道中规中矩"、差评"没有特色" → 产品没特色
- 消费者对"产品值不值30元"持否定态度

**第三层：消费行为分析**
- 路过愿意进来 → 环境对白领有吸引力
- 消费目的：生活享受

### 诊断结论

**问题不是"知名度不够"**

问题在于：产品力不够，但环境有价值。品牌诉求"第三空间"是对的，但只做了环境，没有做内容和体验。

### 解决方向

1. 放大环境优势，做沙龙/社交场所
2. 增加故事性和体验性项目
3. 让"第三空间"不只是空间，是有内容的场域

### 核心洞察

老板以为问题是"知名度"，实际问题是"产品力和定位错位"。砸钱推广会把人引来，但引来的发现"不过如此"，反而浪费钱。

## 案例2：传统食品企业转型（渠道-客群-产品错配）

### 诊断信息

- 20年传统食品企业，主营线下渠道（超市、便利店）
- 年营收5000万
- 找代运营做电商，投50万推广费，转化率低
- 老板判断：问题是"包装不够年轻化"，想重新设计

### 诊断链条

**第一层：客群-渠道匹配**
- 传统食品客群：年纪偏大，习惯线下购物
- 转线上 = 营销阵地转换
- 问题：线上渠道有没有触达原来的客群？还是投错了人群？

**第二层：产品-客群匹配**
- 需不需要年轻客群？
- 年轻客群能不能接受传统产品？
- 如果产品本身和年轻客群存在认知错配 → 光改包装没用

**第三层：能力匹配**
- 如果需要重新设计产品，企业有没有这个能力？

### 诊断结论

**不是"包装不够年轻化"的问题，是三个问题：**

1. 渠道问题：线上投放有没有触达目标客群？
2. 定位问题：要不要年轻客群？产品要不要变？
3. 能力问题：如果要变，企业能不能做到？

### 解决方向

1. 先回答"要不要年轻客群"这个问题
2. 如果要，产品是否需要调整
3. 如果产品要调整，企业有没有能力做到
4. 如果都要，再考虑包装年轻化

### 核心洞察

投了50万推广费转化率低，可能是投错渠道了——年纪大的人不在线上，或者线上渠道不对。不是包装问题，是渠道-客群-产品三者的匹配问题。

## 案例3：新品牌冷启动（缺少尝试理由）

### 诊断信息

- 健康零食新品牌，主打"0糖0脂0卡"
- 已在小红书种草，找KOC推广
- 销量上不去
- 创始人判断：问题是"内容不够吸引人"，想找更大网红合作

### 诊断链条

**第一层：产品差异化**
- 0糖0脂0卡是健康零食的"入场券"，不是差异化
- 每个健康零食品牌都这么打，消费者已经无感

**第二层：尝试理由**
要让消费者尝试，需要找到触发点：
- **诱惑**：好吃到停不下来
- **焦虑**：吃了不胖、吃了不后悔
- **认同**：我就是这样的人（健康生活方式）
- **理解**：品牌懂我（深夜想吃又怕胖）

**第三层：推广策略**
- 找更大网红 = 更贵成本，不适合初创企业试错
- 应该和小网红合作，从内容创新上试错
- 给自己更多调整空间

### 诊断结论

**不是"内容不够吸引人"，是"没找到尝试理由"。**

0糖0脂0卡是标配，不是卖点。

### 解决方向

1. 从诱惑、焦虑、认同、理解四个维度找到触发点
2. 用小网红低成本试错，找到有效的内容方向
3. 内容有效后再考虑放大投放

### 核心洞察

0糖0脂0卡是标配，不是卖点。需要从诱惑、焦虑、认同、理解四个维度找到触发点。初创阶段应该低成本试错，而不是砸钱找大网红。

## 诊断方法论总结

### 核心原则

1. **先看生意，再看品牌** — 没有生意基础，品牌是无根之木
2. **不是找问题，是找差距** — 问题是负面的，差距是可弥补的
3. **验证假设，不轻信判断** — 老板说的问题，不一定是真正的问题
4. **先定位，再方案** — 不知道问题在哪，给方案也是错的

### 常见误判

| 老板说的问题 | 实际问题 |
|------------|---------|
| 知名度不够 | 产品力和定位错位 |
| 包装不年轻 | 渠道-客群-产品错配 |
| 内容不吸引 | 没找到尝试理由 |
| 价格太高 | 没找到购买理由 |
| 竞争太激烈 | 差异化不足 |

### 诊断闭环

```
我是谁 → 消费者需要谁 → 差距在哪 → 如何缩短差距 → 消费者如何找到我
```
